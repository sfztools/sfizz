Detailed Description

template<class Type>
class sfz::EventEnvelope< Type >

Describes a simple envelope that can be polled in a blockwise manner.

It works by storing "events" in the immediate future and linearly interpolating between these events. This envelope can also transform its incoming target points through a lambda, although the lambda function is applied before the interpolation.

The way to use this class is by repeatedly calling registerEvent and then getBlock to get a block of interpolated values in between the specified events. You should only register events whose timestamps are below the size of the block you will require when calling getBlock.
